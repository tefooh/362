# Project 362 Enterprise Edition

This directory contains enterprise-only features for Project 362.

The code in this directory is licensed under the [Project 362 Commercial License](../LICENSE.md), the same license as the rest of the repository. You may use this code for development and testing without a license, but production use requires a valid Project 362 Enterprise subscription or commercial agreement.

## What's in here

- **License key authentication** - replaces user login for IT-managed deployments
- **Admin policy enforcement** - centralized lockable settings pushed by admins
- **Enterprise update management** - Intune/MDM detection and Project 362-managed update policy
- **Enterprise UI controls** - hides consumer UI (billing, login, updates) in managed environments
- **Project 362 SDK** - embeddable capture SDK for Electron, Swift, Tauri, and Node apps in [`sdk`](./sdk)

## What's NOT in here (stays open source)

- Team management, encrypted pipe sync, cloud sync - these are consumer features available to everyone
- Screen recording, OCR, transcription, pipes, search - core functionality, free for personal use
